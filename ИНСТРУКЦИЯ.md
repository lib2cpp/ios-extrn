# Chess Bot — Safari Extension для iOS
## Полная инструкция по установке

---

## Что тебе понадобится
- Windows ПК
- iPhone с iOS 15+
- Apple ID (бесплатный)
- AltStore (бесплатно)
- Xcode — нужен Mac. Но есть обходной путь ниже.

---

## Вариант А — если есть доступ к Mac (хоть раз)

### Шаг 1 — Установи Xcode
Открой App Store на Mac → найди Xcode → установи (бесплатно, ~12 ГБ).

### Шаг 2 — Создай проект расширения
1. Открой Xcode → File → New → Project
2. Выбери платформу: **iOS**
3. Выбери шаблон: **Safari Extension App**
4. Product Name: `ChessBot`
5. Team: выбери свой Apple ID (добавь через Xcode → Preferences → Accounts)
6. Bundle Identifier: `com.yourname.chessbot`
7. Нажми Next → создай проект

### Шаг 3 — Замени файлы расширения
В Xcode открой папку `ChessBot Extension/Resources/`
Удали все файлы внутри и замени на файлы из этой папки:
- `manifest.json`
- `background.js`
- `content.js`
- `popup.html`
- Папку `stockfish/` (скачай отдельно — см. ниже)

### Шаг 4 — Скачай Stockfish WASM
Перейди на: https://github.com/lichess-org/stockfish.wasm/releases
Скачай последний релиз, возьми файлы:
- `stockfish.js`
- `stockfish.wasm`
Положи их в `Resources/stockfish/`

### Шаг 5 — Подпиши и собери
1. В Xcode выбери свой iPhone как target устройство
2. Product → Build (⌘B)
3. Product → Run (⌘R) — установится на iPhone

### Шаг 6 — Включи расширение на iPhone
Настройки → Safari → Расширения → Chess Bot → Включить
Разреши доступ к chess.com

## Вариант Б — только Windows (через AltStore)

Для этого варианта нужен уже собранный `.ipa` файл.
Собрать `.ipa` без Mac нельзя напрямую, но можно использовать:

### Онлайн-сборка через GitHub Actions (бесплатно)

1. Зарегистрируйся на https://github.com (бесплатно)
2. Создай новый репозиторий
3. Загрузи все файлы из этой папки
4. Создай файл `.github/workflows/build.yml` (шаблон ниже)
5. GitHub автоматически соберёт `.ipa`
6. Скачай готовый `.ipa` из раздела Actions → Artifacts

### Файл .github/workflows/build.yml:
```yaml
name: Build IPA
on: [push]
jobs:
  build:
    runs-on: macos-latest
    steps:
      - uses: actions/checkout@v3
      - name: Build
        run: |
          xcodebuild -project ChessBot.xcodeproj \
            -scheme ChessBot \
            -configuration Release \
            -archivePath build/ChessBot.xcarchive \
            archive CODE_SIGN_IDENTITY="" CODE_SIGNING_REQUIRED=NO
          xcodebuild -exportArchive \
            -archivePath build/ChessBot.xcarchive \
            -exportPath build/ipa \
            -exportOptionsPlist ExportOptions.plist
      - uses: actions/upload-artifact@v3
        with:
          name: ChessBot-IPA
          path: build/ipa/*.ipa
```

## Установка через AltStore (Windows)

### Шаг 1 — Установи AltServer на Windows
1. Скачай AltServer: https://altstore.io
2. Скачай iTunes с сайта Apple (НЕ из Microsoft Store):
   https://www.apple.com/itunes/download/win64
3. Скачай iCloud с сайта Apple (НЕ из Microsoft Store):
   https://updates.cdn-apple.com/2020/windows/001-39935-20200911-1A70AA56-F448-11EA-8CC0-99D41950005E/iCloudSetup.exe
4. Установи iTunes и iCloud, затем AltServer
5. Запусти AltServer — появится иконка в трее

### Шаг 2 — Установи AltStore на iPhone
1. Подключи iPhone к ПК по USB
2. Нажми иконку AltServer в трее → Install AltStore → выбери iPhone
3. Введи Apple ID и пароль
4. На iPhone: Настройки → Основные → VPN и управление устройством
5. Нажми на свой Apple ID → Доверять

### Шаг 3 — Установи Chess Bot
1. Перекинь `ChessBot.ipa` на iPhone (AirDrop / Google Drive / Telegram себе)
2. Открой файл на iPhone → "Открыть в AltStore"
3. AltStore подпишет и установит

### Шаг 4 — Включи расширение
Настройки → Safari → Расширения → Chess Bot → Включить ✓
Разреши: "Разрешить на всех сайтах" для chess.com

## Как пользоваться

1. Открой Safari → зайди на chess.com
2. Начни партию
3. Золотая стрелка покажет лучший ход
4. Кликай фигуру сам
5. Настройки: нажми иконку расширения в адресной строке Safari

---

## Переподпись (каждые 7 дней)

Бесплатный Apple ID требует переподписи раз в 7 дней.
AltStore делает это автоматически пока:
- AltServer запущен на Windows ПК
- iPhone и ПК в одной Wi-Fi сети

Или переустанови вручную повторив Шаг 3.

---

## Проблемы

**Стрелка не появляется** — зайди в Настройки → Safari → Расширения → Chess Bot → убедись что включено и разрешён доступ к chess.com

**"Не доверенный разработчик"** — Настройки → Основные → VPN и управление устройством → твой Apple ID → Доверять

**AltStore не видит iPhone** — убедись что iTunes и iCloud установлены с сайта Apple, а не из Microsoft Store
